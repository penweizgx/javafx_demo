# 用户头部下拉菜单

## 概述

将 Shell 顶部区域最右侧的用户名显示，改为"用户名 + 头像（占位）"的组合，并添加下拉菜单，提供"查看登录信息"和"退出登录"两个操作项。

## 动机

当前用户名仅作为纯文本显示，点击后弹出一个 Popover 显示用户详情，交互方式不够直观。需要：

1. **视觉增强**：增加头像占位符，让用户区域更醒目、更符合常见 UI 模式
2. **操作入口**：通过下拉菜单提供明确的操作选项，而非隐藏在点击事件中
3. **退出登录**：当前缺少退出登录功能，用户无法安全退出并返回登录页

## 目标

- 用户区域显示：头像占位 + 用户名
- 点击用户区域弹出下拉菜单，包含：
  - "查看登录信息"：显示当前用户详细信息
  - "退出登录"：清理 Token、关闭主界面、返回登录窗口
- 下拉菜单文本支持 i18n
